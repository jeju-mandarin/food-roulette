# 🌀 오늘 뭐 먹지? 룰렛

"오늘 뭐 먹지?"라는 고민을 해결해 주는 간단한 메뉴 추천 룰렛 웹 애플리케이션입니다.

`localStorage`를 사용하여 사용자가 자신만의 메뉴 리스트를 직접 관리(추가/삭제)할 수 있습니다.

🔗 **[지금 바로 사용해보기 (Live Demo)](https://jeju-mandarin.github.io/food-roulette)**
*(위 링크의 `food-roulette` 부분은 실제 생성하신 저장소 이름으로 변경해주세요.)*

---

## ✨ 주요 기능

* **랜덤 메뉴 추천:** 룰렛을 돌려 메뉴 리스트에서 무작위로 하나의 메뉴를 선택합니다.
* **스핀 애니메이션:** 룰렛이 돌아가는 듯한 간단한 시각 효과를 제공합니다.
* **나만의 메뉴 관리 (LocalStorage):**
    * **메뉴 추가:** 원하는 메뉴를 리스트에 새로 추가할 수 있습니다.
    * **메뉴 삭제:** 리스트에서 원하지 않는 메뉴를 개별적으로 삭제할 수 있습니다.
    * **데이터 영구 저장:** 추가/삭제한 메뉴 리스트는 브라우저(`localStorage`)에 저장되어, 창을 닫거나 새로고침해도 유지됩니다.
* **커스텀 스크롤바:** 메뉴 목록이 길어져도 스크롤 영역임을 쉽게 인지할 수 있도록 스크롤바가 항상 표시됩니다.
